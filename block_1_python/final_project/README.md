# Трекер тестовых сценариев

Консольное приложение для отслеживания результатов тестовых сценариев,
написанное в рамках изучения Python для автоматизации тестирования.

## Возможности
- Хранение тестовых сценариев с результатами по шагам
- Автоматический подсчёт статистики (пройдено/провалено/% успешности)
- Формирование текстового отчёта
- Проверка доступности внешнего сайта через requests

## Технологии
- Python 3.11+
- requests

## Запуск
\`\`\`bash
pip install -r requirements.txt
cd block_1_python/final_project
python main.py
\`\`\`

## Структура проекта
- `test_scenario.py` — класс TestScenario
- `report_builder.py` — функции формирования отчётов
- `site_checker.py` — проверка доступности сайта
- `main.py` — точка входа
